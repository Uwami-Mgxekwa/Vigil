# Vigil

Vigil is a lightweight terminal-based system monitor that provides real-time insights into CPU, memory, and network usage. It helps developers and power users keep track of system performance with alerts when thresholds are exceeded.

## Features

- Real-time monitoring of CPU, memory, and network usage
- Configurable threshold alerts
- Terminal dashboard interface using `tui-rs`
- Cross-platform support (Linux, macOS, Windows)
- Minimal resource overhead

## Installation

### Prerequisites
- Rust (latest stable version). Install via [rustup](https://rustup.rs).
